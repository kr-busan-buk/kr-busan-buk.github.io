# 북구 · 부산광역시

> 북구 지역 정보 랜딩 사이트

## 🌐 랜딩 페이지

### **https://kr-busan-buk.github.io/**

부산광역시 북구는 낙동강 물길과 금정산 자락이 맞닿는 강변 도심으로, 오랜 세월 '구포나루'를 중심으로 사람과 물자가 모여든 교역의 고장입니다. 조선 후기부터 이어진 구포장(구포시장)은 낙동강 유역 곡물과 생활물자가 집산되던 상권의 심장이었고, 오늘날에도 400년 전통의 재래시장으로 활기를 잇고 있습니다. 화명·덕천·만덕 일대는 1990년대 이후 대규모 아파트 신도시로 탈바꿈해 30만 명 가까운 주민이 사는 부산 서북부의 대표 주거지로 성장했습니다. 강변에는 화명생태공원과 금빛노을브릿지가 펼쳐지고, 금정산 기슭에는 부산 최초의 공립 화명수목원이 자리해 도심 속 자연 휴식처가 풍부합니다. 구포국수와 구포만세운동으로 상징되는 서민의 삶과 저항의 역사가 켜켜이 쌓인, 이야기가 있는 동네입니다.
